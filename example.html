Add to Calendar Button

This example is using twitter bootstrap, you can have any library/css to build the dropdown component.

Note: This code uses Vanilla JS methods and doesn't have any Polyfills for cross browser compatibility. Methods that would require a Polyfill are document.querySelectorAll, addEventListener, addEventListener.
